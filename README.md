# currency_cli

Check currency rates against USD

Run cli app

```
./currency_cli
```

Convert value from one currency to another

```
./currency_cli -c kes-ugx 1000
```

## Build executable

### Rust

```shell
chmod +x build-rust.sh
```

```shell
bash build-rust.sh
```

### Go

```shell
chmod +x build-go.sh
```

```shell
bash build-go.sh
```
